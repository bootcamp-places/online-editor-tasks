# returnCollections

Треба реалізувати функцію, яка отримує масив рандомних цілих чисел, а повинна повернути масив, який складається з послідовних колекцій цих чисел.

**Приклад:**

```js
returnCollections([2, 3, 1, 5, 6, 0]); //[[0, 1, 2, 3], [5, 6]]
```
