# uniteUnique

Необхідно реалізувати функцію `unitUnique`, яка приймає в якості аргументу один або декілька масивів , і повертає новий масив
унікальних значень.

**Приклад:**

```js
// [код прикладу використання]
uniteUnique([1,2,3], [4,1,5], [6,7,8,5]); // [1,2,3,4,5,6,7,8]
```
