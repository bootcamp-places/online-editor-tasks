# toUpperCase

Необхідно реалізувати функцію `toUpperCase`, яка приймає в якості аргументу стрічку, а
повертає нову стрічку у верхньому регістрі.

**Приклад:**

```js
toUpperCase('hello'); // 'HELLO'
```
