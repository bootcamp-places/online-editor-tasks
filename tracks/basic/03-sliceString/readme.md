# sliceString

Необхідно реалізувати функцію `sliceString`, яка приймає в якості першого аргументу - 
стрічку, а в якості другого - число символів які треба відрізати від стрічки.

**Приклад:**

```js
sliceString('some long string', 4); // 'some'
```
